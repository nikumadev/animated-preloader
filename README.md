# Preloader Animation - NikumaDev

A lightweight, animated SVG preloader for websites, built with HTML, CSS (SCSS), and Vanilla JavaScript. Smoothly displays a loading animation before showing the main content.

![Animated Preloader](./assets/preloader.png)

## Features :rocket:

- Full-screen, centered preloader animation
- Smooth SVG path animation with fade-in effect
- Prevents scrolling while loading (`noscroll` class)
- Fully responsive on all screen sizes
- Lightweight and easy to integrate into any project

## Installation :memo:

1. Clone or download the repository.
2. Include the preloader in your HTML:

```html
<div class="preloader loader">
  <svg viewBox="0 0 14 10" fill="none" xmlns="http://www.w3.org/2000/svg">
    <path d="..." fill="white" />
  </svg>
</div>
```

3. Link the CSS and JS files:

```html
<link rel="stylesheet" href="./styles/style.css" />
<script src="./scripts/script.js"></script>
```
